## Project Description

A comprehensive AI-powered web application for detecting malicious URLs, phishing attempts, and fraudulent links using Google Gemini AI. The system provides intelligent threat detection with detailed explanations, confidence scores, and comprehensive analysis across multiple attack vectors.

## Key Features

- AI-Powered URL Analysis
- Phishing Detection
- Malware Detection
- Batch URL Processing
- QR Code Analysis
- Threat Intelligence
- PDF/CSV Export
- Real-time Monitoring
- Email Security Analysis

## Technology Stack

- **Backend**: Python 3.7+, Flask 2.3.3
- **AI**: Google Gemini 2.5 Flash
- **Frontend**: Bootstrap 5, Vanilla JavaScript
- **Libraries**: BeautifulSoup4, ReportLab, tldextract, validators

## Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Configure API keys in `.env` file
4. Run the application: `python run.py`
5. Access at `http://localhost:5001`
